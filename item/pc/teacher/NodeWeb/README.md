
#目录结构

#   www.js是服务器的创建
#   app.js是服务器处理程序
#  router文件夹是路由模块
#  config是数据库配置模块（mysql和redis）
#  db在这里就是MVC中的M，用于数据处理
#  controller是MVC中的C，用户数据与视图的衔接处理
#   model文件夹这里只是用于处理响应的数据，是数据模型
